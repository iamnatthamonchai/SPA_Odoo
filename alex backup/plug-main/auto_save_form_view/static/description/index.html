Try to imagine when you input a lot of information on the form view, and then suddenly your browser freezes, you cannot do anything on the browser, and you must restart the browser or in another case your power is shutted down. In that situation, you recognize your form hasn't been saved and you really lost your data. With this app, it will help you save your form automatically at a particular time, you avoid losing your data in case.


List features:

1. Set a particular time to save form view automatically.
After install this app. Default of particular time is 30 seconds. Go to Setting --> Technical --> System Parameters and change "auto_save_timer_key" parameter to apply new particular time saving the form.


2. Allow to save new form automatically or not
Go to Genernal Settings


3. Allow to save a form has any state automatically

If this option isn't selected, then only form has draft state will be saved automatically.


4. If a form hasn't state attribute. It always will be saved automatically.
