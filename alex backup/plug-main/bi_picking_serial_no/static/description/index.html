Generate-Create serial/lot number from the Picking/Receipt Odoo Apps

This odoo apps helps to generate serial number and lot number from picking/receipt, It is very useful to generate custom serial number or lot number from the purchase order automatically when received the goods.This odoo apps generate automatic serial number and lot number from incoming shipment/receipt order when product is received from the purchase order/vendor. This serial/lot number sequence comes with prefix and sequence defined on the configuration settings.

Features

User can set digit.

Users are allowed to add digit as serial number.

User can set prefix.

Users are allowed to add prefix as serial number.

Generates serial/lot number.

You don't have to create serial/lot number manually,it will create number automatically as per configuration like global or product wise.

Generates serial/lot number as per configuration.

It will create new serial/lot no for the product in purchase order as per your input.

Serial Number Configuration

Under Purchase > Configuration > Settings menu you can see serial no configurations.

You have options to set Serial Number.

1. Global Option :

If you select "Global" option then in picking, Product lot/serial number is set based on global digit and prefix.

2. Product Wise Option :

If you select "Product Wise" option then in picking, Product lot/serial number is set based on particular product digit and prefix.


Create Purchase Order

Create purchase order and select product and click confirm order.


Confirm Purchase Order

On click of confirm order picking orders will be created.


Products with Serial No.

You can see the finished products with your custom serial/lot no in picking order.


Product wise Serial Number

Under Purchase > Configuration > Settings menu you can see serial no, Select Product wise option.


Set Serial No.

When you select Product wise serial option, Select products and set digit or prefix on Inventory tab.


Create Purchase Order

Create purchase order and select product and click confirm order.


Products with Serial No.

You can see the finished products with your custom serial/lot no in picking order.
