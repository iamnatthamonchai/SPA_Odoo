Purchase Multi Location | Purchase Putaway Strategy | Direct location from PO

Features:

This app helps you to directly define the destinaton location from purchase order.

Configuration

No congfiguration needed. Install app and in purchase order line you are able to select location. This will directly transfer the goods from vendor to selected location. If no location selected then the default flow.

Once we validate purchase order related incoming shipment created but here the destination location is as per operation types. But if you try to allocate the qty it automatically select the destination location from purchase order line.

If we check products the correct destination location set per lines.
