About
============
Majorly in store, there are many products so sometimes it's difficult to find a particular product so using this module you can find products according to tags in POS. When you type a tag into the search bar, all related products for that tag are displayed.

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list. 
5) search module name and hit install button.
