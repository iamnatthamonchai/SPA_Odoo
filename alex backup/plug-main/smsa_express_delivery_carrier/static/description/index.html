Perks of Integrating your Odoo with SMSA Express
When you are into product selling business then you need to take care of the safe delivery of the products to the customers which can be a really hectic task for you to manage manually. Odoo allows you to perfectly manage the delivery of products so that your customers can get the products without any hassle. But, isn't it great to connect your Odoo with one of the best Shipping Carriers that is SMSA Express Shipping.
Know what Odoo SMSA Express Shipping Integration Does
If you have a business in Saudi Arabia or cater to Saudi Arabian customers on your Odoo website, then integratingSMSA Express with your Odoo website is a feasible solution to your logistics requirements. The module Odoo SMSA Express Shipping Integration allows you to connect your Odoo with SMSA Express Shipping Solution.
Choose various shipping Price Computation Methods
The module offers three different types of shipping price computation methods to choose from.
Fetch live price directly from SMSA Express
Set a fixed price based on rules such as distance, weight, etc.
Add additional commission to the shipping price to earn revenue
  • The module provides different shipping price computation options for you.
  • The methods work for both website orders placed by customers and the orders placed by admin in the Odoo backend.
  • Each option is independent of another and only calculation method can be chosen at a time.

Detailed Features List

Works for both Odoo backend and website

  • The Odoo user can add the SMSA Express shipping to an order from the backend.
  • The customers can choose SMSA Express while placing orders on the website.

Supports Domestic

  • The module will work for domestic shipments of Saudi Arabia

Real-Time Shipment Generation

  • The Shipping is generated when the delivery is validated in the Odoo backend.
  • Generate SMSA Express shipping labels in real-time with a click of a button.
  • The module also generates the SMSA Express shipment tracking number for the order in the Odoo backend.
  • The shipment can also be cancelled from Odoo backend itself.

Set Default Values For Faster Operation

  • Use the default configuration tab to set default values for a number of fields to save time.
  • These values are automatically picked while generating SMSA Express shipment in Odoo. So, you don't need to enter the same values each time.
  • If needed, the values of desired fields can be changed directly from the delivery shipment form.
  • Default values for fields such as product weight, UoM, SMSA Express packaging, etc. can be set.

One-Step Integration With Odoo

  • Simple configuration to allow fast integration of SMSA Express with Odoo.
  • You need Development Username and Password; and Production Username and Password from SMSA Express shipping.
