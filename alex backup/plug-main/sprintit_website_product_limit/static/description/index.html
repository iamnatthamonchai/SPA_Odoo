SprintIT Website Product Limit

This module allows to set a limit how many products can be purchased per one order.

Table of contents

APPLICATION HIGHLIGHTS
FUNCTIONALITY

 

APPLICATION HIGHLIGHTS [back]

Easily set a limit how many pieces of products can be ordered in Webshop.
Set the feature active and define the limit how many pieces of each product can be ordered per Order.

 

FUNCTIONALITY [back]

Go to Product View and set the Order Limit "True", then input the limit for this product.

Webshop will inform the Customer about the limit.

A bigger quantity than the limit can not be set.
